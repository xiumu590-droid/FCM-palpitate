 教程:将脚本移动到已下 路径:
 ```
 /data/local/tmp/ 
 ```
 然后执行:
 ```
 cd /data/local/tmp/ && nohup ./FCM.sh &
 ```
 终止进程指令:
 ```
 pkill -15 FCM.sh
 ```
